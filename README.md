# PrimeProductTable
CLI tool to print a multiplication table of the first n primes.

# Running the Code
1.  Download the release at: https://github.com/TrnsltLife/PrimeProductTable/releases/tag/v1.0
2.  Note: You need Java installed to run this
3.  Unzip the code somewhere
4.  Open a command prompt and navigate to the PrimeProductTable/bin/ directory
5.  At the command prompt type:
    1.  Windows: `PrimeProductTable.bat`
    2.  Linux: `./PrimeProductTable`
